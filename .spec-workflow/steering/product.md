# 产品概述

## 产品目标

QFNU-API-GO 是一个使用 Go 语言封装的曲阜师范大学教务系统 API 中间件。它将教务系统的网页数据转换为结构化的 JSON API，方便开发者在其他应用中调用教务系统数据。

## 目标用户

1. **开发者**: 需要获取教务系统数据用于开发自定义应用（如成绩查询机器人、课表小程序等）
2. **学生**: 通过内置的 Web 界面直接查询成绩、课表等信息
3. **技术爱好者**: 学习 Go 语言 Web 开发和爬虫技术

## 核心功能

1. **成绩查询**: 获取学生各学期成绩数据，支持结构化输出
2. **课表查询**: 获取学生课程表信息
3. **登录认证**: 支持教务系统 Cookie 认证，提供统一的鉴权机制
4. **Web 界面**: 提供简洁的前端页面，可直接在浏览器中使用

## 业务目标

- 提供稳定可靠的教务系统数据接口
- 简化教务系统数据获取流程
- 作为 Go 语言学习项目的参考实现
- 探索极简架构设计的可行性

## 成功指标

- **稳定性**: API 可用率 > 99%（取决于教务系统本身）
- **响应速度**: 单次请求响应时间 < 5 秒
- **易用性**: 开发者可在 5 分钟内完成部署并调用 API

## 产品原则

1. **简洁至上**: 代码和设计保持极简，拒绝过度工程
2. **容错优先**: 对未知输入宽容处理，不轻易报错
3. **单文件部署**: 编译产物为单一可执行文件，开箱即用
4. **够用就好**: 不为抽象而抽象，不为扩展而过度设计

## 监控与可见性

- **界面类型**: 内置 Web 界面（embed 嵌入）
- **日志系统**: 带颜色的控制台日志 + 文件日志（自动切割）
- **关键指标**: 请求日志记录请求路径、响应状态、耗时

## 未来展望

### 潜在增强

- **更多接口**: 支持空闲教室查询、考试安排等更多教务功能
- **缓存机制**: 添加请求缓存减少对教务系统的压力
- **API 文档**: 自动生成 Swagger/OpenAPI 文档
- **Docker 支持**: 提供 Dockerfile 方便容器化部署

## 声明

本项目仅供学习交流使用，请勿用于商业用途。
