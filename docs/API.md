# API 调用文档

## 成绩查询接口

### 接口说明

该接口用于查询学生的课程成绩。用户需要提供有效的 Cookie 以进行身份验证。

### 请求方式

- 方法: `POST`
- URL: `/api/v1/grades`
- 请求头:
  - `Content-Type: application/json`
  - `Authorization: <用户的 Cookie>`

### 请求参数

| 参数名称    | 类型   | 必填 | 说明     | 备注               |
| ----------- | ------ | ---- | -------- | ------------------ |
| term        | string | 否   | 学期     | 例如 "2023-2024-1" |
| courseType  | string | 否   | 课程性质 | 详见下方           |
| courseName  | string | 否   | 课程名称 | 例如 "高等数学"    |
| displayType | string | 否   | 显示方式 | 详见下方           |

### 课程性质说明

支持中文名称或参数值两种方式传递。后端会统一转换为参数值。

| 参数值 | 说明                     |
| ------ | ------------------------ |
| 01     | 公共课                   |
| 02     | 公共基础课               |
| 03     | 专业基础课               |
| 04     | 专业课                   |
| 05     | 专业选修课               |
| 06     | 公共选修课               |
| 07     | 专业任选课               |
| 08     | 实践教学环节             |
| 09     | 公共任选课               |
| 10     | 教师教育基础课程（必修） |
| 11     | 专业必修课               |
| 12     | 学科基础必修课           |
| 13     | 专业方向限选课           |
| 14     | 考试报名虚拟课程         |
| 15     | 教师教育选修课程         |
| 16     | 公共必修课               |

### 显示方式说明

| 参数值 | 说明         |
| ------ | ------------ |
| all    | 显示全部成绩 |
| max    | 显示最好成绩 |
